## ✨ 遊戲特色

1. **🎨 互動式大廳與隊伍選擇**：
   - 玩家可於大廳下拉選單中自行挑選加入 `紅隊 🔴`、`藍隊 🔵`、`綠隊 🟢` 或 `黃隊 🟡`。
   - 支援「加入電腦 (Bots)」一鍵補滿人數，隨時開啟戰局。

2. **🗺️ 動態戰場地圖 (ASCII 渲染)**：
   - 每回合即時渲染 ASCII 戰場地圖，自動繪製所有存活玩家當前位置的隊伍顏色 Emoji，讓戰局一目了然：
     ```text
               [紅隊基地(🔴)]
                    │
     [綠隊基地] ── [鑽石 A] ── [中心島(🔵)] ── [鑽石 B] ── [黃隊基地]
                    │
               [藍隊基地]
     ```

3. **🧠 三種電腦 (AI) 難度**：
   - **簡單**：隨機性大，屬於休閒難度。
   - **普通**：懂得多樣裝備升級，殘血時會購買並吃金蘋果恢復生命值。
   - **困難 (強敵)**：具備警報機制（敵人打入自家基地時會立刻回防）、戰術火球砸牆破防、優先獵殺已碎床隊伍（Final Kill 獵手）、防禦包床（黑曜石優先），非常具有挑戰性！

4. **🛒 分類鐵匠鋪商店**：
   - 捨棄擁擠的下拉選單，全新重構為 4 大功能按鈕分頁：`⚔️ 武器鐵匠`、`🛡️ 防具工具`、`🧱 建造床防禦`、`🧪 特殊道具`，玩家點選後可於個人面板無縫操作。

5. **⌛ 原生相對倒數計時**：
   - 整合 Discord 原生相對時間計時器，各回合會在訊息內自動顯示動態秒數倒數，緊張感滿分！

6. **🧹 自動與手動訊息清理**：
   - 遊戲結束或手動取消時，會自動在背景刪除大廳與回合的冗餘訊息，保持頻道清爽。
   - 若有殘留訊息，可使用指令 `.cw clear` 清空頻道內 100 條機器人訊息。

## 🚀 快速開始

### 1. 安裝需求環境
本專案基於 Python 與 `discord.py` 庫，請先確保已安裝 Python 3.8+，並執行以下指令安裝依賴項目：
```bash
pip install -r requirements.txt
```

### 2. 進行 Bot 設定
在專案根目錄下建立一個名為 `.env` 的設定檔，並填入您的 Discord Bot Token：
```env
DISCORD_TOKEN=您的_DISCORD_BOT_TOKEN
```

### 3. 啟動機器人
執行 `bot.py` 啟動專案：
```bash
python bot.py
```
